Eyebrow + extrabold headline (with optional cyan accent word) + description — the standard section intro.

```jsx
<SectionHeading
  eyebrow="Clinic & Lab Services"
  title="Everything Under"
  accent="One Roof"
  description="Consultations, lab tests, imaging and ECG — all in one accredited facility."
/>
```

`accent` is appended to the title in cyan. `centered` centers everything and adds a trailing eyebrow rule. `light` flips text to white for the navy surface. `as` sets the heading level (use `h1` for the hero).

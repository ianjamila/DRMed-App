Small fully-rounded status pill — result states, counts, metadata chips.

```jsx
<Badge variant="success">Released</Badge>
<Badge variant="cyan" dot>10 HMO Partners</Badge>
<Badge variant="warning">Pending payment</Badge>
```

Variants: `default` (navy), `secondary` (tint), `cyan` (accent), `success` (emerald — released/paid), `danger`, `warning`, `outline`. Pass `dot` for a leading status dot.
